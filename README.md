# Adobe Experience Platform 학습 자료

이 저장소는 Adobe Experience Platform(AEP) 이해도를 높이기 위한 스터디 자료와 실습 프로젝트를 포함합니다.

## 구조

```
adobe-aep-learn/
├── README.md                    # 프로젝트 개요
├── setup/                       # 환경 설정 가이드
├── docs/                        # 이론 학습 자료
├── samples/                     # 샘플 데이터 및 코드
├── projects/                    # 실습 프로젝트
└── resources/                  # 추가 리소스
```

## 학습 방법

상세한 학습 플랜은 [aep-study-plan.md](./docs/aep-study-plan.md)를 참조하세요.

## 시작하기

1. AEP 계정 및 샌드박스 준비
2. 개발 환경 설정 (`setup/` 디렉토리 참조)
3. Week 1-2부터 순차적으로 학습 시작

## 필수 요구사항

- Adobe Experience Platform Trial 또는 Production 계정
- API 접근 권한
- PostgreSQL 클라이언트 (Query Service 사용 시)
- Node.js (Web SDK 실습 시)

## 문의

학습 중 궁금한 사항은 각 주차별 디렉토리의 README를 참조하거나, Adobe Experience League 커뮤니티를 활용하세요.
